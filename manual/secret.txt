6. Чтоб скрипт все время работал на сервере используем проект Forever
   https://github.com/foreverjs/forever
   
   Установка:
   [sudo] npm install forever -g
   Для использование на скрипте нужно установить в папке с проектом монітор:
   cd /path/to/your/project
   [sudo] npm install forever-monitor
   Запуск скрипта:
   forever start app.js
   где app.js скрипт который нужно чтоб все время работал.
   Если возникнет ошибка:
    /usr/bin/env: node: No such file or directory
   Для того чтоб пофиксить выполните команду: 
   sudo ln -s "$(which nodejs)" /usr/bin/node

Удобно устанавливать nodejs определенной версии при помощи NVM.

Устанвливаем сам nvm

Установка пакетов для nvm
sudo apt-get update
sudo apt-get install build-essential libssl-dev


Скачивание пакета для установки nvm
curl https://raw.githubusercontent.com/creationix/nvm/v0.16.1/install.sh | sh

та команда загрузит и запустит скрипт. ПО будет установлено в Вашу домашнюю директорию в папку ~/.nvm. Также в файл ~/.profile будут добавлены некоторые необходимые для работы настройки.

Для получения доступа к функционалу nvm, Вам необходимо перелогиниться в системе, либо Вы можете использовать команду source для того, чтобы применить изменения не прерывая текущую сессию:

source ~/.profile

Чтобы узнать, какие версии Node.js доступны для установки, наберите:

nvm ls-remote

nvm install <номер версии>
nvm use <номер версии>
Посмотреть текущую версию nvm -v
Посмотреть скаченые версии nvm ls

nvm alias default <версия по умолчанию>
